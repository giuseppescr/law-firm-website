# Law Firm Website

## Project Description
This is an institutional website for a professional service, built to demonstrate **Front-end development skills**, **responsive design**, and **client-side interactivity**.

The project ensures a functional, clear, and professional communication platform using modern web standards.

## Key Features & Technologies
* **Front-end Development:** Built with a clean **HTML5** structure and **CSS3** styling (including Flexbox/Grid).
* **Responsive Design:** Utilizes CSS Media Queries and flexible layouts to ensure full functionality and optimization across desktop and mobile devices.
* **Client-Side Logic:** Uses **JavaScript** (in `contact_logic.js`) for form submission logic and user feedback via the `fetch` API, ensuring an interactive user experience (UX).
* **Code Structure:** The project follows modern best practices by separating logic, styling, and structure into dedicated files (HTML, CSS, JS).

## Technology Stack:
* HTML5
* CSS3 (Flexbox/Grid, Media Queries)
* JavaScript (Client-Side Logic)
